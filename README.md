# setup_tools_createNewWorkspace

Minimal TypeScript Hello World project.

## Setup

Install dependencies:

```bash
npm install
```

Build:

```bash
npm run build
```

Run:

```bash
npm start
```

For development without building (requires `ts-node`):

```bash
npm run dev
```

Run tests:

```bash
npm test
```
